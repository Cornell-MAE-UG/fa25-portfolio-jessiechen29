# Problem Description

Given a 2D design space of 150cm long and 50cm tall, a rigid bar of a fixed length (your choice), 3 pin supports of which two need to be mounted on the ground and a linear actuator (pick from this [online catalog](https://www.tolomatic.com/wp-content/uploads/2022/05/2700-4000_29_IMA_cat.pdf), use max force values only), design a frame/mechanism to lift the maximum possible weight to the highest possible height.
Assume all the supports and bar/actuator are rigid.

# Design Degree of Freedom

In this problem, we have one system which can generate three equations (Fx, Fy and M), but we have bar length, lifting height, mass, and reactions at the two supports which are more variables that we can solve with three equations, so for the sake of solving this problem, I will set lifting height to the maximum allowable heigth which is the hight of the design space (50cm) and try to optimize the other variables with h_max as a given quantity.
